# ft-compliance-test
The LonMark FT Compliance Test repository provides a Neuron C application for testing twisted-pair transceiver compliance with the ISO/IEC 14908-2 twisted pair communication standard for the LonMark TP/FT-10 channel type.  This application implements the test requirements described in the LonMark ISO/IEC 14908-2 Compliance Test document, dated 6-May-2026.
